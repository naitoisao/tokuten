📋 このプロンプトの使い方


    
▶ まず試すなら（毎回コピペ）

    
💬

      ChatGPT・Claude・Gemini 共通：

      チャット画面を開く → プロンプトをそのまま貼り付け → 続けて「〇〇を書いてください」と送信。無料プランでOKです。
    


    

    
▶ 一度設定すれば毎回コピペ不要（すべて無料）


    
🤖

      Claude（無料）→ プロジェクト機能

      
    
        
  1. claude.ai を開く
    2. 
        
  2. 左メニューの「プロジェクト」→「プロジェクトを作成」
    3. 
        
  3. 「プロジェクトの指示」欄にプロンプトを貼り付けて保存
    4. 
        
  4. このプロジェクト内でチャットすると自動で反映されます
    5. 
      

    


    
✨

      Gemini（無料）→ Gems機能

      
    
        
  1. gemini.google.com を開く
    2. 
        
  2. 左メニュー「Gems」→「新しいGemを作成」
    3. 
        
  3. 「指示」欄にプロンプトを貼り付けて保存
    4. 
        
  4. 次回からこのGemを選ぶだけでOK
    5. 
      

    


    

    
▶ 有料プランの場合


    
💬

      ChatGPT Plus → GPTs

      
    
        
  1. chatgpt.com を開く
    2. 
        
  2. 「GPTを探す」→「GPTを作成」
    3. 
        
  3. 「Instructions」欄にプロンプトを貼り付けて保存
    4. 
        
  4. ※GPT作成はPlusプラン（有料）のみ
    5. 
      

    


    

    
🔧

      精度UP：出力が「自分っぽくない」と感じたら「もっと〇〇な感じで」「この表現は使わない」と追加で伝えると精度が上がります
